Drop walkthrough screenshots into this folder tree.

How it works:
- Docs pages are already wired to paths like /docs-images/products/<file>.png
- If a file exists, it is shown automatically.
- If missing, the docs show a placeholder card with the expected path.

Recommended image guidelines:
- Format: PNG
- Width: 1600px (or close)
- Include key UI context, not cropped too tightly
- Avoid personal or sensitive account data in captures
